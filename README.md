# RayTracer
Projet EPITECH de 2eme année : Réaliser un programme capable de charger des libraries dynamiques avec un niveau d'abstraction. Jeux réalisé : Snake / Nibbler

## Requirements

`Build`
- Makefile
- make

`Libraries`
- SDL

## Build

```
make
```

## Librairies
### Graphic
- SDL

### Game
- Nibbler
- Snake
